Privacy policy for the web-extension Channel Blocker by Time Machine Development

Last updated: August 12, 2020

The web-extension "Channel Blocker" collects no data of any user! All user data is processed on the users local machine and is never sent to any server!

We would also like to state why the web-extension requires it's permissions:

  • Access browser tabs ("tabs"):
    • needed for keeping track of the YouTube™-Tabs and the Add-on setting page tab
  • Store unlimited amount of client-side data ("storage", "unlimitedStorage"):
    • needed for blocking an unlimited number of users/channels
  • Access to the ContextMenus ("contextMenus"):
    • needed for adding the bug-reüprt button to the contextmenu
  • Access your data for www.youtube.com ("*://www.youtube.com/*"):
    • needed for removing users/channels you have blocked on YouTube™, adding the 'x'-button, etc.
